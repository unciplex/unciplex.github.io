MML后台网管常用指令集

MML网管平台常用命令


1.小区参数

MOD CELLSEL

最小接入电平 default：-64dBm（2dBm）


2.频率级参数

MOD CELLRESEL

  1. (源)小区重选优先级
  2. 小区重选迟滞（dB）; //default：2dB/4dB
  3. 异频/异系统测量启动门限（2dB）; //default：12
  4. 服务频点低优先级重选门限（2dB）; //default：8
  5. 同频测量启动门限（2dB）; //28


3.事件级参数

MOD EUTRANINTERNFREQ

EUTRAN INTER N FREQ (接入网 异频 相邻 频点)

  1. 异频频点小区重选优先级;
  2. 频率偏置（dB）; //default：0
  3. 异频频点高优先级重选门限（2dB）; //default：13
  4. 异频频点低优先级重选门限（2dB）; //default：9



4.（两两）小区重选优先级

MOD EUTRANINTERFREQNCELL

EUTRAN INTER FREQ N CELL (接入网异频相邻邻区信息)

  1. (目标)小区偏置（dB）; // default = 0
